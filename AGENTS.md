# Agent Instructions (Project Level)

This repository did not previously include a project-level agent instruction file.  
`AGENTS.md` is now the canonical project-level instruction entry point for AI agents.

## Read First

1. `AGENT_SUMMARY.md` - fast orientation to architecture, workflows, and key files.
2. `AGENT_STANDARDS.md` - required working standards, safety boundaries, and validation expectations.

## Scope

These instructions apply to the whole repository unless a deeper `AGENTS.md` is added in a subdirectory in the future.
